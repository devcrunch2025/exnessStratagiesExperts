OnTick() — FULL EXECUTION FLOW Runs every price tick
STEP 1 · dxb_ResetDailyLimitsIfNewDay()
Checks if today's date ≠ last stored date.
If new day → resets dxb_g_dailyProfit=0, dxb_g_dailyLoss=0, dxb_g_dailyLimitHit=false.
Panel shows "Status: ACTIVE" again each morning.
STEP 2 · DAILY LIMIT GATE
IF dxb_Use_Daily_Limits=true AND dxb_g_dailyLimitHit=true
→ RETURN — EA silent for rest of day.
Triggered when daily profit ≥ $100 OR daily loss ≥ $50.
STEP 3 · FLOATING LOSS GATE
Calls dxb_GetFloatingLoss() — sums all negative open P&L.
IF float loss ≥ dxb_Floating_Loss_Limit_Per_Symbol ($50)
→ dxb_CloseAllTrades() then RETURN. Emergency exit.
STEP 4 · TRADE COUNT GATE
IF dxb_Enable_Trade_Limits=true:
Wins ≥ dxb_Max_Winning_Trades (5) OR Losses ≥ dxb_Max_Losing_Trades (3)
→ RETURN — session limit reached.
STEP 5 · MANUAL RESET CHECK
IF dxb_Set_TRUE_to_Reset_All_Limits=true:
Clears all counters and flags — unlocks a stopped EA.
User manually sets this in the Inputs panel.
STEP 6 · SPREAD FILTER
Reads live spread via MarketInfo(MODE_SPREAD).
IF spread > dxb_MaximumSpread (20 pts) → RETURN.
Prevents trading during news spikes with wide spreads.
STEP 7 · NEW BAR FILTER
IF Time[0] == dxb_g_lastBarTime → RETURN.
Locks to one action per M1 candle open. Prevents multi-entry on same bar.
Updates dxb_g_lastBarTime = Time[0] on pass.
STEP 8 · dxb_ManageOpenTrades()
Runs profit protection on ALL open trades BEFORE new entries.
Tracks dxb_g_highestProfit. Applies MODE 1/2/3/4 if enabled.
Step lock: if profit drops to locked floor → close all and exit.
STEP 9 · OPEN TRADE BRANCH
IF dxb_CountOpenTrades() > 0:
→ Calls dxb_CheckAdditionalEntry() — grid/averaging logic.
IF price moved ≥ 50 pts against trade → open same-direction entry. RETURN.
STEP 10 · dxb_GetSARSignal()
No open trades → look for new signal.
Reads Parabolic SAR (step=0.0014, maxstep=0.0126).
SAR flips below price → BUY (1) | SAR flips above price → SELL (-1) | No flip → 0 (RETURN)
STEP 11 · dxb_GetCurrentLotSize()
Auto: Base(0.01) + TradeCount × Increment(0.01), capped at 0.10.
Custom: reads Trade1..Trade10 array, cycles with modulo.
Both normalize to broker's min/max/step requirements.
STEP 12 · dxb_OpenTrade(signal, lot)
Sends OrderSend() BUY or SELL at market.
SL = entry ± 530 points. TP = 0 (managed by EA, not broker).
On success: dxb_g_tradeCount++, stores ticket number.

Profit Protection — 4 Modes Master switch must be ON · Activates after $5 profit
M1
BREAK-EVEN + LOCK
Trigger: total profit ≥ Mode1_Lock_Profit ($1)
Action: moves SL of ALL trades to OrderOpenPrice()
Result: guaranteed $0 loss if market reverses
Direction: BUY SL moves up · SELL SL moves down
Default: DISABLED
M2
TRAILING STOP
Trail Distance: $3 behind current price
BUY: new SL = Bid − trail (rises with price)
SELL: new SL = Ask + trail (falls with price)
Rule: SL only improves — never widens
Default: DISABLED
M3
PARTIAL CLOSE
Close %: 50% of each profitable trade's volume
e.g. 0.10 lot → closes 0.05 lot · keeps 0.05 open
Filter: only applies to trades with profit > 0
Result: banks partial gain, stays in the market
Default: DISABLED
M4
STEP LOCK ★ RECOMMENDED
Tracks highest profit ever reached (dxb_g_highestProfit)
As profit grows through steps, floor ratchets up:
Profit $5 → Lock $1 | Profit $10 → Lock $5
Profit $15 → Lock $10 | Profit $20 → Lock $15
IF profit drops to locked floor → CLOSE ALL TRADES
Default: ENABLED ✓
